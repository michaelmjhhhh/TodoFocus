# TodoFocus Workflow Checklist

## Bugfix Path
- [ ] Reproduce bug with explicit steps
- [ ] Capture root cause evidence from code/logs/tests
- [ ] Add or update failing test when practical
- [ ] Implement single focused fix
- [ ] Re-run targeted and full verification

## Feature Path
- [ ] Create/update issue markdown
- [ ] Create issue and branch
- [ ] Write plan markdown if work is non-trivial
- [ ] Implement in small coherent commits
- [ ] Verify tests/build before PR

## PR Completion
- [ ] PR markdown includes `Closes #<issue>`
- [ ] PR markdown includes verification commands and results
- [ ] Only relevant files are staged
- [ ] Branch pushed and PR created from markdown body
