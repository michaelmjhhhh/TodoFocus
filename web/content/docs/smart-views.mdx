# Smart Views & Search

TodoFocus gives you multiple ways to find tasks — from quick filters to full-text search.

## Smart Views

In the sidebar, you'll find preset views that automatically filter your task list:

| View | What it shows |
|------|---------------|
| **My Day** | Tasks flagged for today — the things you've committed to working on |
| **Important** | Tasks marked with `!` — high priority |
| **Planned** | Tasks with any due date set |
| **Overdue** | Tasks past their due date |
| **All** | Every task, regardless of state |
| **Archive** | Archived tasks |
| **[Custom Lists]** | Your own lists with optional color |

### My Day

Marking a task as "My Day" is a signal to yourself: this is what I'm working on today. It's not just a filter — it's a commitment.

You can add or remove tasks from My Day directly from the task row or task detail view.

### Custom Lists

Create your own lists in the sidebar:

- **Name** — give it a descriptive title
- **Color** — pick a color to distinguish it visually; a colored bar appears on the left edge of task rows

Lists appear in the sidebar and filter to just tasks in that list.

## Time Filters

Each view supports a **time filter chip** at the top of the task list:

- **All** — no time filter
- **Today**
- **This Week**
- **Next Week**

Switch between them to narrow down what's relevant right now.

## Search

Press **`⌘K`** to open the search bar. TodoFocus searches across:

- Task **titles**
- Task **notes** (including appended Quick Capture notes)

Results appear instantly as you type.

## Kanban: Open vs. Completed

The main task list has two panels:

- **Open** — active tasks matching the current view and filter
- **Completed** — tasks you've marked Done; collapsible to keep the view clean

## Archiving

Tasks can be **archived** (moved to the Archive view) or **deleted** permanently:

- **Archive** — hides the task but keeps it recoverable
- **Clear Completed** — archives all completed tasks at once
- **Empty Archive** — permanently deletes all archived tasks
