# Daily Review

TodoFocus's Daily Review is a Kanban-style board that gives you a clear, honest view of where everything stands. Not analytics. Not dashboards. Not guilt. Just: what needs to move?

## Opening Daily Review

- Press **`⌘⇧R`**
- Or click **Daily Review** in the sidebar

The board shows two horizontal lanes: **Open** and **Completed**.

## The Columns

Within each lane, tasks are grouped by time horizon:

| Column | What goes here |
|--------|----------------|
| **Overdue** | Tasks past their due date |
| **Today** | Tasks due today |
| **Tomorrow** | Tasks due tomorrow |
| **Next 7** | Tasks due within the next week |
| **No Date** | Tasks with no due date |
| **Done** | Tasks marked complete this session |

## Card Actions

Each task card in the board supports quick actions:

- **Move to Today** — reschedule to today
- **Move to Tomorrow** — reschedule to tomorrow
- **Move to Next 7** — reschedule within the week
- **Remove Date** — clear the due date
- **My Day** — toggle the My Day flag
- **Done** — mark complete and move to Done column

## Open vs. Completed Lanes

- **Open** — all unfinished tasks across time horizons
- **Completed** — tasks marked Done this session, visible in a collapsible panel

The Completed panel can be collapsed for a cleaner view focused on what's still open.

## How It Feels

Open Daily Review at the end of your workday:

1. Scan **Overdue** — either do it now or reschedule it
2. Sweep **Today** — anything not done today either moves to tomorrow or gets a hard look
3. Check **Tomorrow** — make sure tomorrow's plan still makes sense
4. Move **Done** tasks out of the way

A five-minute ritual that replaces the vague anxiety of "what should I be doing?" with a concrete, honest picture.

## Keyboard Reference

| Shortcut | Action |
|----------|--------|
| `⌘⇧R` | Open Daily Review |
| `Esc` | Close Daily Review |
