## Summary
Add a manual Daily Review workflow entry in Sidebar for reviewing all tasks and taking quick cleanup actions.

## User Requirements
- Entry point: Sidebar
- Scope: review all tasks
- Actions: mark done, reschedule, add to My Day
- Trigger rhythm: manual only
- Outcome: show explicit review summary stats

## Success Criteria
- User can open Daily Review from Sidebar.
- Review list includes all current tasks.
- User can run core cleanup actions quickly.
- Summary counters update during the review session.
